# Online Bildung Information System

Das **Online Bildung Information System** ist ein **Schulprojekt**.

Dieses Projekt wurde für ein Schulprojekt entwickelt.  
Das Ziel ist ein **digitaler Verifikations-Service** für Dokumente in Deutschland.

In Deutschland gibt es für viele Dokumente **keine digitale Prüfung**.  
Dieses Projekt bietet eine **digitale Lösung**.

---

## Ziel des Projekts

- Offizielle Dokumente digital prüfen
- Dokumente einfacher und schneller nutzen
- Weniger Papier benutzen

---

## Beispiel Dokument

In diesem Projekt gibt es ein Beispiel:

- **Schülerbescheinigung**

Dieses Dokument ist **digital**.

---

## Wie funktioniert es?

- Das System erstellt ein Dokument
- Das Dokument hat eine **digitale Signatur**
- Das Dokument ist **offiziell und gültig**
- Keine extra Unterschrift ist nötig

---

## Test Information

Dieses Projekt kann man **nur in meiner Schule testen**.

- Es gibt ein **spezielles Gerät**
- In diesem Gerät ist der **Verifikations-Schlüssel**
- Ohne dieses Gerät **funktioniert das System nicht**
